antidiagonals(grid) → Array of Array of T

Gets diagonals of a grid in the other direction to grid_utils.diagonals (so top left to bottom right). Assumes the grid is rectangular.

Example 

antidiagonals([
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]) // [[3], [2, 6], [1, 5, 9], [4, 8], [7]]
